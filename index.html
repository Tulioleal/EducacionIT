Atributos de una Etiqueta de Bloque

  • Margin: Distancia que existe entre etiquetas (No suma tamaño y siempre aplica el mayor)
  • Border: Linea que rodea la etiqueta (suma al tamaño Final)
  • Padding: Distancia que existe entre el borde y el contenido de una etiqueta (suma al tamaño Final)
  • Content: Elementos contenidos por una etiqueta html (podemos definirle un tamaño mediante los atributos WIDTH y HEIGHT)

Etiqueta de Linea que no posee carga semantica

  • inline CSS: atributo style="reglas" - Mas Especifico
  • block CSS: etiqueta <style> </style> - Menos Especifico (General - Afecta a toda la Pagina)
  • CSS file: Hoja de estilos vinculada mediante LINK - General (Puede afectar a multiples Paginas)
