Formularios HTML

Son herramientas que permiten una interaccion real (intercambio de datos) entre el cliente y el servidor, ya que a diferencia de otros elementos con los que el usuario puede interactuar, el resultado de esta interaccion dependera de la finalidad del mismo. Por ejemplo, podemos utilizar un formulario para OBTENER datos de un servidor como es el caso de los buscadores; asi como tambien ENVIAR informacion a traves del servidor a una base de datos o por correo electronico.

¿Cuantos Tipos existen?

Segun los atributos que se utilicen, popularmente podemos clasificarlos en dos grandes grupos. Antes que nada es fundamental utilizar la etiqueta FORM para encerrar los campos del mismo y que el navegador entienda que es lo que se debe enviar, a la vez que dentro de esta etiqueta indiqueta el valor para el atributo METHOD que determina la finalidad del formulario.
Los valores para este atributo son GET , el cual es menos seguro, prioriza los datos recibidos por parte del servidor y los datos enviados se pueden visualizar en la barra de direcciones y admite hasta 8kb. Por otro lado existe el metodo POST , que se utiliza para darle prioridad a los datos enviados, ya que cifra los mismos, es mas seguro y admite hasta 8mb por envio.

  • GET
    • Buscadores - Consulta Query
  • POST
    • Inscripcion - Registro
    • Acceso - Inicio Sesion
    • Consulta Web - Contacto

¿Como estan compuestos?

Dentro de FORM utilizaremos dos tipos de etiquetas, segun su clasificacion. Por un lado tenemos las etiquetas (LABEL) que se utilizan como guia para que el usuario sepa que informacion debe colocar en el formulario, asi como tambien para los Lectores de Pantalla (screen readers) ya que estos no leen el otro tipo de etiquetas. La otra categoria de elementos es el campo (INPUT - TEXTAREA - SELECT) , el cual permite al usuario seleccionar o escribir la informacion deseada, por parte del formulario.

  • Texto (Aa123!#)
  • Numero (123)
  • Email (usuario@dominio)
  • Contraseña (***)
  • Fecha (dd-mm-aaaa)
  • Campo de opcion Unica
  • Campo de opcion Multiple
  • Reestrablece todos los campos dentro del mismo FORM
  • Ejecuta el ACTION del FORM actual
  • Cuadro Multilinea
  • Lista Desplegable
